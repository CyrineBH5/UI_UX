# Digital Minds

## Description
Digital Minds est un projet web réalisé dans un cadre académique.
Il a pour objectif de développer une interface moderne, responsive et structurée.

## Technologies utilisées
- HTML5
- CSS3
- Tailwind CSS (via CDN) // SCSS
- JavaScript

## Exécution du projet
Aucune installation requise.

1. Ouvrir le fichier `index.html`
2. Lancer directement dans un navigateur (Chrome, Edge, Firefox)
